# bi0s CTF Write-ups

Welcome to my bi0s CTF repository! This repo contains write-ups and solutions to various challenges I have solved while participating in bi0s CTF competitions. These write-ups are meant to help others understand the techniques used in solving the challenges.

## About bi0s CTF

[bi0s](https://bi0s.in) is India's No. 1 ethical hacking team, and they frequently host challenging and exciting Capture The Flag (CTF) competitions. This repo documents my journey through these competitions and highlights my learning process.
